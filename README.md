# Tog Refusjon
Basert på ClaimPilot-arkitekturen (Next.js + Firebase).
Fullautomatisert løsning for togrefusjon og forsinkelseskrav.

## Videre utvikling
- Frontend bygges i Next.js/React (webapp/PWA)
- Backend bygges på Firebase (Auth, Firestore, Cloud Functions)
- Integrasjon mot Entur/Bane NOR for sanntidsdata
- Automatisert kravbrev og refusjonshåndtering
